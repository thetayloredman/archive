Learn JS (Lesson 1 - Using the Console)

What is the console?

The console is the output (STDOUT) of your program. It's where everything that your program outputs goes.

Using the console

To use the console, we will use what's called console.log. Here's an example of writing to the console:

console.log("Hello, World"); // => Outputs "Hello, World" on the console
