Learn JS (Lesson 2 - Basic Data Types)

What are "data types"?

A data type is a type of data that can be represented in your script. Examples of data types are:

String // (set of words)
Number // (Number, well a number)
Array // (A list)
Object // (A set of key-value pairs)

What are the JavaScript data types and how do I use them?

Here's a list of data types we will cover:

String
Number
Array
Object

Now using them, is a bit odd. Each data type has it's own "symbol" we use to replicate it:

String // " or '
Number // just a number
Array // [index1,index2]
Object // {key:value,another:key}

Using this syntax is simple! To log a number, we log:

console.log(123);

To log a word, we use:

console.log('Hello, World!');

To log an array, we use:

console.log(['a','b','c']);

Etc.
